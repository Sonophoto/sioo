NAME

version - SiOO version

SYNOPSIS

version

DESCRIPTION

This command gives information about the current SiOO version being run. The string returned is formated as follow:

PROCESS_NAME XX.XX.XX.XX [devo|alpha|beta|gamma|release]

Where PROCESS_NAME is the string returned in argv[0][]
XX.XX.XX.XX is returned by cmd_version() [release type] is returned by cmd_version()

EXAMPLE

None at this time
:wq
